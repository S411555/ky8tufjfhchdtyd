# Student Management System (MySQL Project)

## Description
This project is a simple database system for managing students, courses, and their results.

## Features
- Add students
- Add courses
- Enroll students in courses
- Store and update marks
- Simple joins for reporting

## Tables
- students
- courses
- enrollment

## Technologies Used
- MySQL

## How to Run
1. Open MySQL Workbench / XAMPP
2. Run the SQL file
3. Execute queries step by step
